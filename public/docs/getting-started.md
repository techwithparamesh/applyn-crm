# Getting Started

## Overview

Applyn CRM is a flexible customer relationship management platform that helps you manage leads, deals, contacts, and workflows in one place. This guide will help you get up and running quickly.

## Navigation

The main application is accessed after signing in. Use the **sidebar** on the left to move between areas:

- **Dashboard** — Home view with metrics and widgets
- **Modules** — Your data types (Leads, Contacts, Deals, etc.)
- **Templates** — Prebuilt CRM setups
- **Pipelines** — Sales or process stages
- **Relationships** — Links between modules
- **Communication** — Email and WhatsApp
- **Workspace** — Tasks, Automations, Forms, Reports
- **Integrations** — Third-party connections

Use the **Help** button (?) in the header to open this documentation anytime.

## UI Components

- **Sidebar** — Collapsible navigation; click the trigger to collapse/expand.
- **Global search (⌘K)** — Search records and modules from anywhere.
- **Header** — Notifications, profile, and Help.

## User Actions

- Sign in / Sign out
- Open any section from the sidebar
- Use global search to jump to records or modules
- Open Help to read documentation

## Step-by-Step Guide

### How to get started

1. Sign in to your workspace.
2. Open **Dashboard** to see your overview.
3. Go to **Modules** and open or create a module (e.g. Leads).
4. Create a few records to see how data entry works.
5. Optionally install a **Template** for a ready-made structure.
6. Use **Help** (?) in the header whenever you need guidance.

## Best Practices

- Start with one or two modules and add more as needed.
- Use **Templates** to get a proven structure (e.g. sales pipeline) quickly.
- Keep field names and stages consistent so reports and automations stay simple.
- Use the **Help** center and search to find feature-specific guides.
