# Settings: Workspace

## Overview

Workspace settings control the identity and regional behavior of your CRM: workspace name, subdomain, timezone, currency, and date format. These affect how data is displayed and how links and exports look.

## Navigation

**Sidebar → Settings** (or user menu → Settings) → **General** tab.

## UI Components

- **Workspace name** — Display name of your organization/workspace.
- **Subdomain** — Used for workspace-specific URLs (e.g. `yourcompany.applyn.app`).
- **Timezone** — Default timezone for dates and times.
- **Currency** — Symbol and format for money fields.
- **Date format** — How dates are shown (e.g. MM/DD/YYYY or DD/MM/YYYY).

## User Actions

- Edit workspace name and subdomain
- Select timezone, currency, and date format
- Save changes (applies across the workspace)

## Step-by-Step Guide

### How to change workspace name

1. Open **Settings → General**.
2. Update **Workspace name**.
3. Click **Save** (or **Save settings**).

### How to set timezone and currency

1. Open **Settings → General**.
2. Choose **Timezone** and **Currency** from the dropdowns.
3. Save; new and existing data will use these for display and reports.

## Best Practices

- Use a clear workspace name that matches your company or product.
- Set timezone to your primary office so deadlines and activity make sense.
- Pick one currency for the workspace if most deals are in that currency.
