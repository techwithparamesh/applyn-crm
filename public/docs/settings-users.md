# Settings: Users

## Overview

User management lets you invite people to your workspace, list active users, and deactivate accounts. Invited users receive an email to join; once in, you can assign them to teams and roles to control what they can see and do.

## Navigation

**Sidebar → Settings → Users** tab.

## UI Components

- **User list** — Name, email, role(s), status (active/invited).
- **Invite user** — Email and optional role; sends invitation.
- **Actions** — Edit role, deactivate, resend invite.

## User Actions

- Invite users by email (with optional role)
- View and search users
- Change a user’s role
- Deactivate a user (they lose access)
- Resend invitation email

## Step-by-Step Guide

### How to invite a user

1. Open **Settings → Users**.
2. Click **Invite user** (or **Add user**).
3. Enter **Email** and optionally select **Role**.
4. Send; the user receives an email to sign up or join the workspace.

### How to change a user’s role

1. Open **Settings → Users**.
2. Find the user and click **Edit** or the role dropdown.
3. Select the new **Role** and save; permissions update immediately.

## Best Practices

- Assign the least privilege needed (e.g. role with limited modules).
- Use **Teams** for grouping and **Roles** for permission sets.
- Deactivate users when they leave instead of deleting, to preserve audit history.
