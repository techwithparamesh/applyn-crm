# Settings: Teams

## Overview

Teams group users (e.g. Sales, Support, Marketing). You can use teams for assignment (e.g. assign records to a team), reporting (e.g. by team), and access control. Create teams and add members from the Settings → Teams tab.

## Navigation

**Sidebar → Settings → Teams** tab.

## UI Components

- **Team list** — Name, member count, optional description.
- **Create team** — Name and description.
- **Team detail** — Members list; add or remove members.
- **Add member** — Select user(s) to add to the team.

## User Actions

- Create teams
- Add and remove members
- Edit team name and description
- Delete a team (members are not deleted)

## Step-by-Step Guide

### How to create a team

1. Open **Settings → Teams**.
2. Click **New team** (or **Create team**).
3. Enter **Name** and optional **Description**.
4. Save; the team appears in the list. Open it to add members.

### How to add members to a team

1. Open **Settings → Teams** and select the team.
2. Click **Add member** (or **Invite to team**).
3. Select one or more users and confirm; they appear in the team’s member list.

## Best Practices

- Create teams that match how you work (e.g. by department or region).
- Use teams in reports and filters to see performance by group.
- Combine with **Roles** so team members get the right permissions.
