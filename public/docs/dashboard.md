# Dashboard

## Overview

The Dashboard is your home screen. It shows key metrics, charts, and quick actions so you can see performance and jump into work without opening multiple pages.

## Navigation

**Sidebar → Dashboard** (first item, or click the app logo/home).

## UI Components

- **Metric cards** — Show values such as Total Leads, Active Deals, Revenue, Conversion Rate, Records Growth.
- **Charts** — Line charts (e.g. records over time), bar charts (e.g. pipeline distribution), pie charts.
- **Table widgets** — Small tables for recent or summary data.
- **Quick actions** — Shortcuts to create records, run automations, or open common pages.
- **Recent activity** — Timeline of recent changes or events.

Widgets can be added, removed, and rearranged. Use the widget menu (edit/maximize/delete) on each card to customize.

## User Actions

- View metrics and trends at a glance
- Add, remove, or reorder dashboard widgets
- Click quick actions to create records or open features
- Resize or maximize widgets for focus

## Step-by-Step Guide

### How to add a widget

1. Open **Dashboard**.
2. Click **Add widget** or use the dashboard edit mode.
3. Choose widget type (metric card, bar chart, pie chart, line chart, table, quick actions).
4. Configure title, data source, and options.
5. Save; the widget appears on your dashboard.

### How to view pipeline distribution

1. Open **Dashboard**.
2. Find the **Pipeline distribution** (or similar) chart widget.
3. Use filters or date range if the widget supports them.
4. Hover for exact values.

## Best Practices

- Put the 3–5 metrics you check daily at the top.
- Use one pipeline chart so deal stages are visible without opening Pipelines.
- Add a **Recent activity** widget to stay on top of changes.
- Keep the layout simple so the dashboard loads fast and stays readable.
