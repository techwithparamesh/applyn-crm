# Integrations

## Overview

Integrations connect Applyn CRM with external tools and services—email providers, WhatsApp, analytics, productivity apps, and more. The Integrations page lists available integrations, connection status, and lets you connect or disconnect accounts.

## Navigation

**Sidebar → Integrations**.

## UI Components

- **Integration cards** — Name, description, status (Connected / Not connected).
- **Connect** — Start OAuth or setup flow for the service.
- **Disconnect** — Remove the connection (may revoke access on the provider side).
- **Configure** — Optional settings (e.g. sync options, mapping).

## User Actions

- Browse available integrations
- Connect an integration (e.g. Gmail, Outlook, WhatsApp, Zapier)
- Disconnect or reconfigure
- View which integrations are active

## Step-by-Step Guide

### How to connect an integration

1. Open **Sidebar → Integrations**.
2. Find the integration (e.g. Gmail, WhatsApp) and click **Connect**.
3. Complete the provider’s sign-in or authorization flow.
4. When done, the card shows **Connected** and you can use the feature (e.g. Email or WhatsApp in the CRM).

### How to disconnect

1. Open **Integrations** and find the connected service.
2. Click **Disconnect** (or **Settings** then **Disconnect**).
3. Confirm; the connection is removed and the feature may stop working until you connect again.

## Best Practices

- Connect only the accounts you need to avoid clutter and security risk.
- Use official OAuth flows; don’t share passwords with third-party apps.
- Disconnect integrations when switching providers or retiring an account.
