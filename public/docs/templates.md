# Templates

## Overview

Templates are prebuilt CRM setups—modules, fields, pipelines, and sometimes sample data. Installing a template gives you a ready-made structure (e.g. sales pipeline, lead management) so you don’t have to create everything from scratch.

## Navigation

**Sidebar → Templates**. Browse available templates and click **Install** to add one to your workspace.

## UI Components

- **Template gallery** — Cards with name, description, and “Install” button.
- **Template detail** — Full description, list of modules and pipelines included, install option.
- **Install flow** — Confirm; template modules and pipelines are created in your workspace.

## User Actions

- Browse available templates
- View template details (modules, pipelines, fields)
- Install a template (creates modules and pipelines; may add sample data)
- Use or customize the created modules and pipelines after install

## Step-by-Step Guide

### How to install a template

1. Open **Sidebar → Templates**.
2. Browse the list and click a template to see details.
3. Click **Install** (or **Use this template**).
4. Confirm; the template’s modules and pipelines are created in your workspace.
5. Go to **Modules** and **Pipelines** to start using or customizing them.

## Best Practices

- Choose a template that matches your process (e.g. sales, support, lead management).
- After installing, review and adjust field names and pipeline stages to match your team.
- You can add more modules or relationships on top of the template.
- Use templates as a starting point; duplicate or modify as your needs change.
