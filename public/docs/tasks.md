# Tasks

## Overview

Tasks help you track to-dos, follow-ups, and deadlines. Create tasks, assign them to users, set due dates, and optionally link them to records (e.g. “Call this lead”). Tasks can be listed and filtered so nothing falls through the cracks.

## Navigation

**Sidebar → Workspace → Tasks**.

## UI Components

- **Task list** — All tasks with title, assignee, due date, status.
- **Filters** — By assignee, status, due date, or linked record.
- **Create task** — Form: title, description, assignee, due date, link to record.
- **Task detail** — Full view and edit; mark complete.

## User Actions

- Create tasks with title, assignee, due date, and optional record link
- Edit and delete tasks
- Mark tasks complete or change status
- Filter by assignee, date, or record
- Open linked record from the task

## Step-by-Step Guide

### How to create a task

1. Open **Sidebar → Workspace → Tasks**.
2. Click **New task** (or **+ Task**).
3. Enter **Title**; optionally set **Assignee**, **Due date**, **Description**, and **Link to record**.
4. Save; the task appears in the list and in the assignee’s view.

### How to assign a task

1. Create or edit a task.
2. In **Assignee**, select a user (or leave unassigned).
3. Save; the assignee sees the task in their task list and may get a notification.

## Best Practices

- Link tasks to records when the task is about a specific lead, contact, or deal.
- Set due dates so you can filter by overdue or upcoming.
- Use a small set of statuses (e.g. To do, In progress, Done) for consistency.
- Combine with **Automations** to create tasks when a record hits a stage or trigger.
