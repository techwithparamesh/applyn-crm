# Reports

## Overview

Reports let you analyze your data with charts and tables. Create reports based on modules, apply filters and groupings, and add them to dashboards or schedule them by email. Use reports to track pipeline value, conversion, activity, and more.

## Navigation

**Sidebar → Workspace → Reports**. Click a report to open it; use **New report** to create one.

## UI Components

- **Report list** — All reports with name, type, and last run or update.
- **Report builder** — Data source (module), chart type (bar, line, pie, table), filters, grouping.
- **Chart / table** — Rendered report with legend and optional drill-down.
- **Filters** — Date range, stage, owner, or other fields.
- **Add to dashboard** — Pin the report as a widget.
- **Schedule** — Send report by email on a schedule (if available).

## User Actions

- Create a report and choose module and chart type
- Apply filters and grouping
- View and export report data
- Add report to a dashboard widget
- Schedule email delivery
- Edit or delete reports

## Step-by-Step Guide

### How to create a report

1. Open **Sidebar → Workspace → Reports**.
2. Click **New report** (or **Create report**).
3. Select **Module** (e.g. Deals) and **Chart type** (e.g. bar, pie, table).
4. Set **Group by** or **Metric** (e.g. by stage, by owner).
5. Apply **Filters** (date range, stage, etc.) if needed.
6. Save; the report runs and displays. Optionally **Add to dashboard** or **Schedule**.

### How to add a report to the dashboard

1. Open the **Report** you want on the dashboard.
2. Click **Add to dashboard** (or **Pin to dashboard**).
3. Choose position or confirm; the report appears as a widget on the Dashboard.

## Best Practices

- Use one primary metric per report so the message is clear.
- Filter by date range so reports reflect current performance.
- Add key reports to the Dashboard for at-a-glance visibility.
- Use grouping (e.g. by stage or owner) to compare segments.
