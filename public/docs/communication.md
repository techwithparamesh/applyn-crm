# Communication

## Overview

The Communication section covers **Email** and **WhatsApp**. Use Email to connect your inbox, send and receive messages, and link threads to records. Use WhatsApp to manage conversations and link contacts to CRM records.

## Navigation

**Sidebar → Communication**:

- **Email** — Inbox, compose, and record-linked threads
- **WhatsApp** — Inbox and chat (when connected)

## UI Components

- **Email:** Inbox list, thread view, composer, link-to-record actions.
- **WhatsApp:** Conversation list, chat window, message input, contact linking.

## User Actions

- **Email:** Connect account, sync messages, compose, reply, link email to a record.
- **WhatsApp:** Connect account, send/receive messages, link contact to a record.

For detailed steps, see **Email** and **WhatsApp** in the documentation sidebar.

## Best Practices

- Link important emails and chats to the right records so context is in one place.
- Use one connected email account per user for clarity in the inbox.
- Keep WhatsApp connected and respond from the CRM to maintain history with contacts.
